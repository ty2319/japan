先制点で吉元を楽にしたい打線は1回表、先頭の梅津が幸先よく鋭い当たりのレフト前ヒットを放ち、その後得点圏にランナーを進めるも、4番高本が空振り三振に倒れ惜しくも得点にはつながらない。

吉元は注目の立ち上がり、先頭バッターにデッドボールを与えてしまい、盗塁を決められてランナーを2塁に背負った場面で、相手4番にしぶとくライト線に運ばれ、先制点を許してしまう。しかし冷静さを失うことなく、後続からはきっちり三振を奪い、最少失点でこの回を切り抜ける。

すぐさま追いつき、追い越したい打線は2回表、5番中西が左中間を深々と破る2ベースを放ち、絶好のチャンスメイク！続く仮谷がきっちりと送りバントを決め、1アウト3塁とする。しかし、不運も重なったまさかのスクイズ失敗でこの場面も同点とはならず…。

4回表、先頭の中村が粘った末にファーストのエラーを誘う強打で出塁し、ノーアウトのランナーを出すも、後続からは快音が響かず得点を挙げることができない。

吉元は尻上がりに調子を上げ、4回までに7つの三振を奪うナイスピッチングを披露！エースの力投に応えたい打線は6回表、先頭の大城があわやホームランかというセンターオーバーの2ベースを放ち、再びのチャンスメイク！しかし、この場面でもあと一本が出ず…。

6回裏にはミス絡みでさらに1点を追加され、0-2とリードを広げられてしまう。
終盤は尾嶋→藤本→松田と小刻みな継投リレーでピンチを切り抜け、味方の反撃を待つも、試合はこのまま0-2で悔しい敗戦となってしまった。

【試合前】　N - 東

10年連続都市対抗本戦出場を果たした北崎選手兼コーチの表彰式が行われた。

【1回表】　N 0 - 0 東

先頭・梅津が幸先よくレフト前ヒットで出塁するも無得点に終わる。

【1回裏】　N 0 - 1 東

送りバントを防ぐ好守備が出るも1点を失う。

【2回表】　N 0 - 1 東

この回先頭の中西が 左中間への2ベースヒットで反撃ののろしを上げるが、スクイズ失敗で無得点。

【3回裏】　N 0 - 1 東

吉元、ノーアウト２塁のピンチも、冷静な投球で無失点に切り抜ける。

【4回表】　N 0 - 1 東

先頭の中村がエラーで出塁するも無得点に終わる。

【6回表】　N 0 - 1 東

先頭・大城がセンターのフェンス最上段を直撃する2ベースでチャンスをつくるも無得点。

【7回裏】　N 0 - 2 東

6回の失点に続き、この回もピンチを迎えるが、2番手の尾嶋が後続を抑え、事なきを得る。

【8回表】　N 0 - 2 東

流れを変えるべく、副主将の長田が代打で登場するが、相手投手を攻略できず。

【8回裏】　N 0 - 2 東

藤本→松田の投手リレーでピンチを脱出。

【9回表】　N 0 - 2 東

最終回、円陣を組んで気持ちを入れなおす。

【試合終了】　N 0 - 2 東

しかし、最後まで相手投手を打ち崩せず、初戦突破ならず。

【試合終了】　N 0 - 2 東

この敗戦を教訓に、今後のチームのさらなる奮起に期待したい。
